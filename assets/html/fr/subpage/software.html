Logiciel

Vous avez une idée de logiciel ou de site mais il vous manque les compétences techniques? Laissez nous vous aider à réaliser votre projet. Notre équipe étant composé entre autre, d'ingénieurs en informatique, nous serons ravis d'apporter une solution technique à votre projet. Nous réalisons des sites pour chacun de nos artistes afin qu'ils se concentrent sur leurs productions, mais également des applications plus complexe. À moins que cela ne soit une volonté de votre pars, nos logiciels et sites sont open source, ce qui signifie que leur code source est public.

Vous pouvz retrouver l'intégralité des code sources de nos logiciels sur notre page d'organisation GitHub
